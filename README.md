# Factory Energy Cost Calculator

## Run on your computer

1. Install Python 3.10 or later.
2. Put `app.py`, `requirements.txt`, and the sample Excel file in the same folder.
3. Open Command Prompt or Terminal in that folder.
4. Run:

```bash
pip install -r requirements.txt
streamlit run app.py
```

The app will open automatically in your web browser.

## Run online with Streamlit Community Cloud

1. Create a GitHub repository.
2. Upload `app.py` and `requirements.txt`.
3. Go to Streamlit Community Cloud.
4. Connect the repository and select `app.py`.
5. Deploy the app.

The finished app will have its own shareable web link.
